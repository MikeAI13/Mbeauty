## Як встановити на телефон

### Android (Chrome)
1. Відкрий сайт у Chrome
2. З'явиться банер «Додати на головний екран» — натисни
3. АБО: меню ⋮ → «Встановити додаток»

### iPhone (Safari)
1. Відкрий сайт у Safari
2. Натисни кнопку «Поділитись» □↑
3. Оберіть «На головний екран»

## Як опублікувати в інтернеті (безкоштовно)

1. **GitHub Pages** — завантаж файли на github.com, увімкни Pages
2. **Netlify** — перетягни папку на netlify.com
3. **Vercel** — vercel.com → імпорт папки

## Функціональність

- 🏠 Головна — hero, акції, топ послуги, майстри, відгуки
- 💅 Каталог Beauty — 8 послуг для дівчат (манікюр, педикюр, вії, брови...)
- ✂️ Каталог Barber — 8 послуг для чоловіків (стрижки, борода, гоління...)
- 📅 Онлайн запис — форма з вибором майстра, дати, часу
- 📍 Контакти — Instagram, телефон, Telegram, адреса
- 📲 PWA — встановлюється як додаток, працює офлайн
- 🔔 Банер встановлення на Android
